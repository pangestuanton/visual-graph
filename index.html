Map Legend

Visual guide to all symbols and elements on the campus topology canvas.

Regular Building

Green circular marker used for regular campus buildings: Gedung A, B, C, D, E, and F. These represent standard faculty or administrative buildings.

GKU Building

Orange square marker used for Gedung Kuliah Umum (GKU) nodes. GKU 1 and GKU 2 are multi-purpose lecture halls that serve as major intersections in the campus graph.

Normal Route

Blue line showing an available weighted route between two buildings. Each route has a distance weight in meters. The graph contains 10 undirected edges total.

Shortest Path

Red highlighted route showing the result of Dijkstra's algorithm. Active nodes along the path are ringed in red and their labels turn red. The path uses a glow effect for emphasis.

120m

Distance Badge

Small label placed near the midpoint of each edge showing the distance weight in meters. Badges are offset perpendicularly from the route line to avoid overlap.

Result Banner

Yellow card with a red "Result" tape label that displays the shortest path and total distance after running Dijkstra's algorithm. Uses the brutalist rotated-card style.

Node Info Card

Displayed when clicking on a building node. Shows the node name, its neighbors, and the distance to each neighbor. Uses an orange "Node Info" tape label.

Edge List Table

Campus Topology Canvas

The main map area with a cream background, decorative green fields, and a dark header bar labeled "Campus Topology Canvas". Nodes, edges, and badges are rendered here.
